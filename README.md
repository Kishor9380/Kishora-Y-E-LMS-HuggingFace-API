# Kishora Y E LMS Assignment

A premium Learning Management System (LMS) platform developed as a professional React application.

## 🚀 Experience the Live App
The project is built using modern web technologies:
- **React 19**
- **Vite 6**
- **Tailwind CSS 4**
- **Lucide Icons**
- **Persistent LocalStorage State**

## 🛠️ Features
- Comprehensive course catalog with search and filtering
- Professional user dashboard with progress tracking
- User authentication (login/register/password reset)
- Dynamic lesson player with YouTube integration
- Interactive certificate generation and PDF download
- Real-time notification system
- Messaging and profile management

## 📦 Deployment Instructions

### How to Run Locally
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

### How to Deploy to Vercel
1. Push this folder to a new **GitHub repository** named `Kishora-Y-E-LMS-Assignment`.
2. Link your GitHub account to **Vercel**.
3. Import the repository in Vercel.
4. Vercel will automatically detect **Vite** settings and deploy your live site!
